层级面板位于编辑器的最左侧，它以树状结构显示当前场景中的所有节点，场景节点是所有其他节点的父节点，包括相机、灯光、网格等等。

<Image src="https://mdn.alipayobjects.com/huamei_fvsq9p/afts/img/A*YOCVRa_kAaAAAAAAAAAAAAAADqiTAQ/original" />

在层级面板，你可以编辑实体的基本属性和管理组件，详见[实体](/docs/core/entity)文档。
