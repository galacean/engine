## 主菜单
通过点击侧边栏的第一个按钮，可以唤起主菜单。主菜单提供了项目设置，新建/克隆项目和一些编辑选项。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/8ea98fe5-399b-4e5f-83c7-391e2e1a36b8/image-20250212120635786.png" />

### 新建/克隆项目

选择 **New Project** 项，可以进一步选择新建不同类型的项目。点击 **Fork**，会跳转到新克隆的项目页面，旧的项目仍会保留。

### 项目设置

点击 **Project Settings** 项，会出现项目基础设置弹窗。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/db9ba443-256a-49da-aaaf-7481b4301009/image-20250515103554317.png"  style={{ zoom: "50%" }} />

项目基础设置包括：

- **Engine Version**：引擎版本升级，以便快速修复某个 bug 或享受新的功能。
- **Physics Backend**：物理引擎后端，可以选择 _Physics Lite_ 或 _PhysX_ 两种后端。前者是一个轻量级的物理引擎，后者是基于 [PhysX](https://developer.nvidia.com/physx-sdk) 的高级物理引擎。
- **Model Import Options**：模型导入选项，包含计算切线、移除灯光的选项。

<Callout type="warning">引擎版本升级操作是不可逆的。为避免损坏项目，升级引擎过程中会自动保存一个快照。</Callout>

为了快速进行项目设置，也可以从项目列表页进行项目设置。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/b1647904-871d-4ec4-9a84-b549f0294610/image-20250515103848864.png"  style={{ zoom: "50%" }} />

## 面板布局

你可以通过此菜单唤起隐藏的面板，也可以根据自己的使用习惯保存当前面板布局，方便其他项目也使用次布局。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/cafce601-36cb-4cf5-9e9c-c4e689bf7de8/image-20250212153709151.png"  style={{ zoom: "50%" }} />

## 成员管理

您可以通过电子邮件搜索新成员，并为其添加读写、只读和编辑三种权限等级。成功添加后，您仍然可以编辑成员的权限等级或移除该成员。此外，您可以生成一个有效期为7天的项目邀请链接，点击该链接的用户可以直接加入项目。

请注意，我们目前不支持多人同时编辑同一个项目。我们将在后续版本中支持多人协作编辑。

<Callout>每个项目最多有 5 名协作者</Callout>

## 快照管理

快照管理功能允许用户保存某个项目的快照到历史记录中，万一项目出现数据丢失等问题。用户可以在菜单中选择 **Add Snapshot** 。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/f1288a4c-0a0b-4f4c-9325-041ba5df0e59/image-20250514175303329.png"  style={{ zoom: "50%" }} />

快照显示了名称、引擎版本、作者等信息，点击快照名可以编辑快照名称，以方便下次快速找到。你可以通过 **Revert** 快速恢复到之前保存的某个快照，也可以通过 **Delete** 删除不想保留的快照。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/041346eb-973e-4e13-aa45-cd1ef29e2b1a/image-20250514175044474.png" style={{ zoom: "50%" }} />

<Callout type="info">如果在项目设置中升级了引擎版本，系统会自动保存一份原先版本的项目的快照。万一升级导致了项目无法打开，你可以从项目列表页的项目设置中恢复到原来的版本。</Callout>
