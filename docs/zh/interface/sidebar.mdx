## 主菜单
通过点击侧边栏的第一个按钮，可以唤起主菜单。主菜单提供了项目设置，新建/克隆项目和一些编辑选项。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/8ea98fe5-399b-4e5f-83c7-391e2e1a36b8/image-20250212120635786.png" />

### 新建/克隆项目

选择 **New Project** 项，可以进一步选择新建不同类型的项目。点击 **Fork**，会跳转到新克隆的项目页面，旧的项目仍会保留。

### 项目设置

点击 **Project Settings** 项，会出现项目基础设置弹窗。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/032598c7-cffb-4a7a-ac1f-2dc2185ce794/image-20250212152704057.png"  style={{ zoom: "50%" }} />

项目基础设置包括：

- **Engine Version**：引擎版本升级，以便快速修复某个 bug 或享受新的功能。
- **Physics Backend**：物理引擎后端，可以选择 _Physics Lite_ 或 _PhysX_ 两种后端。前者是一个轻量级的物理引擎，后者是基于 [PhysX](https://developer.nvidia.com/physx-sdk) 的高级物理引擎。
- **Model Import Options**：模型导入选项，包含计算切线、移除灯光的选项。

<Callout type="warning">引擎版本升级操作是不可逆的。为避免损坏项目，升级引擎过程中会自动保存一个快照。</Callout>

## 面板布局

用户可以通过此菜单唤起隐藏的面板，也可以根据自己的使用习惯保存当前面板布局，方便其他项目也使用次布局。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/cafce601-36cb-4cf5-9e9c-c4e689bf7de8/image-20250212153709151.png"  style={{ zoom: "50%" }} />

## 成员管理

用户可以通过邮箱搜索到新成员，添加读写、只读、编辑三种等级的权限。添加成功之后，仍然可以对成员的权限等级进行编辑或者移除该成员。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/e8804737-d181-4cb5-bb39-74bf800264c1/image-20250212154556326.png"  style={{ zoom: "50%" }} />

## 项目导出

详见[多平台导出](/docs/platform/platform)。

## 快照管理

快照管理功能允许用户保存某个项目的快照到历史记录中，万一项目出现数据丢失等问题，可以通过 **Revet** 快速恢复到之前保存的某个快照。用户可以在菜单中选择 **Add Snapshot** 。点击快照名可以编辑快照名称，以方便下次快速找到。

<Image src="https://gw.alipayobjects.com/zos/OasisHub/c057ba9a-d757-4ab2-8123-9fff84aaeae3/image-20250212121056362.png" style={{ zoom: "50%" }} />
