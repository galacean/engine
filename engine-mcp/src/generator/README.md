# Generator 模块

在 MCP 架构中，**实际的代码生成由大模型完成**，本模块主要负责：

## 核心职责

### 1. 提示词工程 (Prompt Engineering)
- 构建结构化的提示词模板
- 整合上下文信息和知识图谱
- 优化提示词以获得更好的生成效果

### 2. MCP 调用管理 (MCP Call Management)
- 封装对大模型的 MCP 调用
- 管理调用参数和配置
- 处理调用结果和错误

### 3. 后处理 (Post-processing)
- 格式化大模型生成的代码
- 应用代码风格规范
- 添加必要的注释和文档

### 4. 模板管理 (Template Management)
- 管理代码模板库
- 提供模板选择逻辑
- 支持模板的动态组合

## 架构说明

```
用户意图 → 提示词构建 → MCP调用 → 大模型生成 → 后处理 → 最终代码
```

**重要**: 本模块不直接生成代码，而是协调和优化整个生成流程。
